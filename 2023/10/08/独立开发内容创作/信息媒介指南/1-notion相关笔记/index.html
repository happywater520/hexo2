    • 开始使用 Notion | NotionChina

      • 块、页面、数据库、视图、模板templates

      • 界面常用操作 分享、锁定、导出 了解即可

        **Trash**包含所有你删除的页面。你可以把它们拖到这里。你也可以点击 Trash 来搜索、查看和恢复这些页面。

        Add to Favorites 添加到收藏夹: 将该页面钉在你的左侧边栏的顶部。

        • 最后编辑： 查看谁最后在该页上做了修改，以及什么时候。
        • 字数： 页面上的字数统计，方便参考。

        Lock page 锁定页面： 这样任何人都无法进行编辑，除非他们明确关闭该锁，包括你自己。

        **Database Lock 锁定数据库**。该数据库中的每个人（包括你自己）都只能编辑数据，但不能编辑视图、属性或 选择 Select 和 多选 Multi-Select 属性的值。

        只要你邀请某人加入一个私人页面，你就会在你的侧边栏看到**Shared**这个类别。

        **Share to web**被关闭，所以只有你和你的队友可以看到这个页面。

        **Copy link**将页面的唯一 URL 复制到你的剪贴板上，这样你就可以与任何你想要的人分享。但是，只有对该页面有访问权限的人才能看到它。

        默认情况下，你的公共页面将不会出现在搜索引擎的结果中。但是，如果你愿意，打开 **Share to the web 分享到网络，开启Search engine indexing 搜索引擎索引**。（plus用户）

        让人们复制你的页面（或不允许复制）他们会在右上方看到一个**Duplicate 复制**按钮。点击它将会把你的页面完全复制到他们侧边栏的 **Private 私人**部分。点击右上方的 **Share 分享**关掉 **Allow duplicate as template 允许复制为模板**。

        默认情况下，你的工作区的公共页面URL将被赋予一个唯一的字符串作为标识符。如果你更改了你的工作区域名，任何以以前的域名开始的现有链接将不再有效。你将需要重新分享新的链接。**notion.site（plus用户）**

        想把你的数据从 Notion 中导出并保留在你的硬盘上？你可以一次将整个工作区导出为HTML、PDF（仅限企业版）或 Markdown & CSV。

        Export 导出： 将页面作为文件下载到你的电脑。数据库被保存为 CSV 文件，而非数据库页面被保存为 Markdown 文件。你也可以把你的页面或数据库导出为 PDF 或 HTML。
